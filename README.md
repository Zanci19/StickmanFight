# StickmanFight

A simple two-stickman fighting game with Mortal Kombat-inspired flow:

- Basic attacks: punch and sweep
- Special move with stronger knockback
- Takedown cinematic with camera movement, zoom, slow motion, and 3 impact repeats
- Visible hit reactions where body parts recoil on impact

## Run

Open `/home/runner/work/StickmanFight/StickmanFight/index.html` in any modern browser.

## Controls

- **Move:** `A` / `D`
- **Punch:** `J`
- **Sweep:** `K`
- **Special:** `L`
- **Takedown:** `I` (only when enemy is low health and in range)
- **Restart after KO:** `R`
